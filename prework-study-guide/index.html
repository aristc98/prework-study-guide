HTML

  • HTML is used to create structure for webpages and is made up of elements such as: text elements, image elements, header elements, and list elements.
  • Looking at a house, HTML would be like the frames for the house, CSS would be like the style, and JavaScript would be like the things you interact with in the house such as being able to open and close a door.
  • A tag is what's inside the angle brackets and the elements is what's between the opening and closing tags.
